# text-generator-app
 application that generates text based on LSTM Network

## Execution
1. clone this repo
2. in the terminal,execute
```
python3 app.py
```

## Dataset
Shakespeare text corpus [link](https://storage.googleapis.com/download.tensorflow.org/data/shakespeare.txt)
